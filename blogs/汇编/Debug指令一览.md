## 寄存器

### 查看

```assembly
r 
# 查看所有寄存器
r <寄存器>
# 查看指定寄存器
```

### 修改

```assembly
r <寄存器>
# 回车后可输入指定值
```



## 存储器

### 查看

```assembly
d 段地址:起始偏移地址 [<结尾偏移地址>] [<L范围>]

# d DS:0
#查看从DS:0开始的内存单元
# d DS:0 5
# 查看从DS:0到DS:5的内存单元
```

### 修改

```assembly
e 段地址:起始偏移地址 修改值 修改值 ……
```
